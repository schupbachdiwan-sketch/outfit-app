# 设计规范

> 版本：v1.0 | 更新日期：2026-05-25

---

## 1. 设计理念

**关键词**：清爽 · 干净 · 年轻 · 直观

- 视觉风格：扁平化插画，无3D效果、无拟物化
- 目标用户：18-35岁年轻群体
- 情感基调：轻松、友好、不给用户压力

---

## 3. 字体规范

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 | 24px | Bold(700) | 32px | 页面大标题 |
| H2 | 20px | SemiBold(600) | 28px | 区域标题 |
| H3 | 16px | SemiBold(600) | 24px | 卡片标题 |
| Body | 15px | Regular(400) | 22px | 正文 |
| Caption | 13px | Regular(400) | 18px | 说明文字 |
| Small | 11px | Regular(400) | 16px | 标签、辅助信息 |

- 中文字体：PingFang SC（iOS）/ 思源黑体（Android）
- 数字/英文：SF Pro Text（iOS）/ Roboto（Android）

## 6. 阴影

```
// 卡片阴影（轻量）
box-shadow: 0px 2px 8px rgba(0, 0, 0, 0.04);

// 浮层阴影
box-shadow: 0px 4px 16px rgba(0, 0, 0, 0.08);
```

原则：阴影克制使用，仅用于区分层级关系，避免厚重投影。

## 7. 图标

- 使用 Feather Icons 或 Phosphor Icons 风格
- 线性图标（outline），描边宽度 1.5px
- 选中态填充版
- 图标尺寸：24px（导航）、20px（行内）、16px（标签内）

## 8. 组件设计

### 8.1 底部导航栏

- 4个Tab：试衣间 | 衣柜 | 灵感 | 我的
- 高度：56px（不含安全区）
- 选中态：主色填充图标 + 主色文字（12px）
- 未选中态：灰色线性图标 + 灰色文字

### 8.2 衣物卡片

- 白色卡片，圆角12px，轻量阴影
- 显示：去背景后的衣物缩略图 + 名称 + 来源标签
- 来源标签：
  - 自有衣物：绿色小圆点 + "衣橱"
  - 心愿单：橙色小圆点 + "心愿"
- 卡片尺寸：宽度自适应（Grid 2列），高度约180px

### 8.3 试衣画布

- 居中展示2D身材模板
- 背景：极浅灰（区分于模板白色）
- 画布支持手势：拖拽、缩放、旋转
- 底部工具栏：撤销 | 重置 | 保存 | 分享

### 8.4 抽屉式衣柜入口

- 从底部向上滑出
- 分类Tab横向滚动（全部 | 上衣 | 下装 | 连衣裙 | 外套 | 配饰 | 鞋 | 包）
- 衣物以2列网格展示
- 长按衣物进入拖拽模式

### 8.5 AI推荐卡片

- 横向滑动卡片列表
- 每张卡片展示完整搭配缩略图
- 卡片底部标注搭配风格说明（如"通勤知性风 · 用了你的3件衣服"）
- 操作按钮：套用 | 保存 | 分享

## 9. 动效规范

| 场景 | 动效 | 时长 |
|------|------|------|
| 页面切换 | 淡入淡出 | 200ms |
| 抽屉展开 | 从底部滑入 ease-out | 300ms |
| 衣物拖入画布 | 弹性缩放到目标位置 | 300ms |
| 卡片点击反馈 | 缩放至95% | 100ms |
| 加载中 | 骨架屏闪烁 | 持续 |

原则：动效轻快、不拖沓，不为了动效而动效。

---

## 10. 空状态与错误状态

| 状态 | 处理方式 |
|------|---------|
| 衣柜为空 | 插画 + "还没有衣服，去添加第一件吧" + 添加按钮 |
| 搭配推荐为空 | 插画 + "衣柜里衣服还不够，先添加几件吧" |
| 网络错误 | Toast提示 + 重试按钮 |
| 加载中 | 骨架屏 |

---

## 11. 身材模板插画规格

| 属性 | 要求 |
|------|------|
| 风格 | 扁平化人物插画，类似矢量UI插画风格 |
| 尺寸 | 800×1600px PNG（含透明通道） |
| 姿势 | 正面站立，手臂自然微张（与身体留间隙便于上衣贴合） |
| 肤色 | 提供3种肤色选项（浅/中/深） |
| 身体分区 | 模板需标注参考线：颈线、肩线、胸线、腰线、臀线、膝线、踝线 |
| 变体 | 5种体型 × 3种肤色 = 15张模板图 |
