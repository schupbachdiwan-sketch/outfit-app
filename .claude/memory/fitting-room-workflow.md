## 试衣间完整流程

### 步骤1：设定身体（二选一）
- 📷 上传全身照 → AI生成用户标准站姿图（穿贴身衣）
- 📐 输入三围（胸/腰/臀）+ 身高体重 → 模特身材随参数动态调整
- 结果：中间假人模特 = 用户的穿衣人台/身材替身

### 步骤2：上传衣服到衣柜
- 📷 拍衣服照片或从相册选
- → AI自动抠图 + 将衣服处理为正面板正的样子
- → 用户给衣服命名
- → 放入指定衣柜文件夹（用户可自定义衣柜名称，如"上班""运动""约会"）

### 步骤3：给模特穿衣服
- 从底部衣柜**长按拖动**衣服到模特身上
- 用户**自己决定**衣服放在哪个位置（肩上/腰上/腿上，不自动吸附）
- 可自由拖动调整位置、缩放大小
- 可叠加多件（内衣→上衣→外套→配饰 层级顺序）

### 步骤4：查看效果（AI生图）
- 点击「查看效果」按钮触发AI
- AI综合三个输入生成最终效果图：
  1. 用户身体数据（照片或三围参数）
  2. 每件衣服的图片
  3. 每件衣服在模特上的位置和大小
- → 输出：用户穿上这套搭配的真实效果图
- → 可保存、分享

### 核心设计原则
- **模特只负责"衣服摆在哪"**，不负责呈现最终效果
- **AI生图是最后一步**，拿着身体+衣服+位置三个数据合成穿上后的样子
- 衣服位置由用户自由决定，不是自动吸附到身体区域
