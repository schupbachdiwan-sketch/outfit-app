## Bug 修复：衣服抠图 "仅支持图片文件"

- **根因**：Flutter Web 的 `http.MultipartRequest` 不传文件分片 `Content-Type`，服务器端拒绝
- **修复**：
  - 服务端 `/api/remove-bg` 支持 JSON base64 模式（Web）+ multipart（原生）
  - 客户端 `removeBackgroundBytes()` 改用 JSON 请求体，与 `generateModel()` 一致
  - 实测通过：0.56s 完成抠图

## 新功能：衣服 AI 增强 v0.6.0

新增端点 `POST /api/enhance-clothing`，流水线：
1. rembg 去背景 → 透明 PNG
2. 纯白(#FFFFFF)底合成 → JPEG
3. Wan2.7 img2img → 专业电商产品图

### 提示词（用户指定）

**正向**：专业电商服装产品图，幽灵模特效果，完全去除人物模特，服装正面完整展示，居中构图，纯白无影无缝背景，无多余杂物，专业影棚均匀柔光打光，无硬阴影无强烈反光，面料纹理清晰真实，版型平整自然，完整还原服装的颜色、印花、剪裁与所有设计细节，8K超高清，边缘锐利干净，商业静物摄影质感

**负向**：模特，人脸，人体，肢体，皮肤，穿在身上的效果，服装褶皱扭曲，版型变形，污渍，水渍，杂乱背景，深色背景，浓重阴影，反光光斑，模糊，低像素，水印，文字logo，侧视角，背面，项链手链帽子等多余配饰，多余人物

### 实测结果

- 耗时：12.14s（rembg + Wan2.7）
- 输出：943KB 高质量 JPEG
- 成本：rembg 免费 + Wan2.7 ~¥0.04/张
